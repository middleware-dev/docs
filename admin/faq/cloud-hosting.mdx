title: Cloud Hosting
---

<Accordion title="What Cloud is Middleware hosted in?">
We default host in AWS and in Azure, and can spin up new regions or in other Clouds as needed. [Reach out to the Middleware team](https://middleware.io/contact-us/) to learn more.
</Accordion>

<Accordion title="Can Middleware be run on-premise?">
Yes, Middleware can be run on-premise and is great for security-conscious companies, or those who want to manage their scale themselves. We do however limit this to enterprise contracts and it requires a minimum spend. [Reach out to the Middleware team](https://middleware.io/contact-us/) to learn more.
</Accordion>


<Accordion title="Can Middleware run on Equinix Bare Metal?">
Yes, Middleware is a verified partner with Equinix and can be deployed as a SaaS platform on Equinix Bare Metal, or on-premise for certain size customers. [Reach out to the Middleware team](https://middleware.io/contact-us/) to learn more.
</Accordion>
