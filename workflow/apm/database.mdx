# APM vs Integration

The APM Database dashboard displays database data that is autocollected by the APM. This dashboard will not display comprehensive information about your dashboard like a [Database Integration]() would. 

# Tags

Tags are the specific properties contained within your database spans. Use the tag feature to search for specific queries, database types, and regex patterns. 

# Query Operation

Query Operations display all of your database queries grouped by request.
